📡 API Reference

Base URL: http://localhost:4000/api

Swagger docs: http://localhost:4000/api/docs

Authentication

All protected endpoints require Authorization: Bearer <token> header.

MethodEndpointDescriptionAuth
POST/auth/registerRegister new userNo
POST/auth/loginLoginNo
GET/auth/profileGet current user🔒 Yes

Books

MethodEndpointDescriptionAuth
GET/booksList books (search/filter/paginate)No
GET/books/:idBook detail with reviewsNo
POST/booksCreate book🔒 Seller
PUT/books/:idUpdate book🔒 Seller (owner)
DELETE/books/:idDelete book🔒 Seller (owner)

Cart

MethodEndpointDescriptionAuth
GET/cartGet cart🔒 Yes
POST/cart/itemsAdd to cart🔒 Yes
PUT/cart/items/:itemIdUpdate quantity🔒 Yes
DELETE/cart/items/:itemIdRemove item🔒 Yes
DELETE/cartClear cart🔒 Yes

Orders

MethodEndpointDescriptionAuth
POST/ordersCreate order from cart🔒 Yes
GET/ordersList my orders🔒 Yes
GET/orders/:idOrder detail🔒 Yes
DELETE/orders/:idCancel order (PENDING only)🔒 Yes

Reviews

MethodEndpointDescriptionAuth
GET/reviews/book/:bookIdList reviews for bookNo
POST/reviews/book/:bookIdAdd review (buyer only)🔒 Yes
PUT/reviews/:idUpdate own review🔒 Yes
DELETE/reviews/:idDelete own review🔒 Yes
GET/reviews/myMy reviews🔒 Yes

HTTP Status Codes

CodeMeaning

201Created
