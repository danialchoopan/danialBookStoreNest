🛠️ Development Guide

Environment Setup

Prerequisites

  • Node.js 18+
  • Docker & Docker Compose
  • Git

Install Dependencies

# Backend
cd backend && npm install

# Frontend
cd frontend && npm install

Coding Conventions

TypeScript

  • Use strict TypeScript (strict: true)
  • Define interfaces in types/ (frontend) or inline (backend)
  • Prefer interface over type for object shapes

Adding a New Feature

Backend (New Endpoint)

  1. Create DTO in modules/feature/dto/
  2. Add to Service in modules/feature/feature.service.ts
  3. Add to Controller in modules/feature/feature.controller.ts
  4. Register Module in app.module.ts
  5. Update Schema if needed: npx prisma migrate dev --name add_feature

Frontend (New Page)

  1. Create page in app/feature/page.tsx
  2. Use 'use client' directive (if using hooks)
  3. Use TanStack Query for data fetching
  4. Add navigation link in Header.tsx if needed

Testing

# Backend unit tests
cd backend && npm run test

# Backend e2e tests
cd backend && npm run test:e2e

# Frontend lint
cd frontend && npm run lint

Git Workflow

Commit Messages

feat(scope): description
fix(scope): description
docs(scope): description
refactor(scope): description

Examples:

feat(books): add advanced filtering
fix(auth): handle expired JWT
docs(api): update endpoint documentation
